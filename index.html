Test Game

by: Wendy

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

The purpose for writing this game is to test the Game Webpage creation process for League Level 2 games so that I will be able to help my students.

2. What I Learned

The release process seemed well documented. The only exception was that the students did not understand that they had to edit the ReadME file to insert the link for their game. I also found out that the color selection for my HTML page does not work on a PC :-(.

3. Architecture of My Game

This game has a runner class, which instantiates the Game Panel. The Game Panel is responsible for the high-level control of the game, but relies on the Object Manager to do the detailed operations. I told my students that they have to write much more than this...

4. Technical Implementation

The game uses a 2D array to create the colored boxes. The Java graphics paint component is used to make the colored rectangles using the Random class to select the colors. A timer was created for test purposes, but not used for the game.

5. Most Challenging Part

This game was a simple test game, and as a teacher I better not have had any difficulties!!

6. Future Fixes and Updates

If I have need to test out new concepts for a game to help a student, I could use this game as a test platform.
